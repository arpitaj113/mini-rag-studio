# 🚀 Mini RAG Studio

AI-powered PDF Question Answering using Retrieval-Augmented Generation (RAG).

## Features

- 📄 Upload PDF
- 🔍 Semantic Search
- 🤖 Google Gemini
- 🗂️ FAISS & ChromaDB
- 📚 Retrieved Chunks
- 📌 Source Pages
- ⚡ Response Time

## Tech Stack

- Python
- Streamlit
- LangChain
- Google Gemini
- FAISS
- ChromaDB
